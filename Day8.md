# 🌍 Day 8 – Personal Environmental Health Analyzer

## 📌 Objective

Build an interactive Environmental Health Analyzer using Claude AI that visualizes environmental data through an engaging HTML dashboard.

---

## 🛠️ Project Overview

For Day 8 of the 60 Days AI Challenge, I used Claude AI to generate a fully functional Environmental Health Analyzer dashboard. The application presents environmental data in an interactive and user-friendly interface with charts, filters, and health insights.

---

## ✨ Dashboard Features

* 🌫️ Air Quality Index (AQI) Monitoring
* 💧 Water Quality Analysis
* 📊 Interactive Charts & Visualizations
* 🎛️ Dynamic Filters
* 📈 Environmental Health Score
* 💡 Personalized Recommendations
* 📱 Responsive HTML Dashboard

---

## 📂 Generated HTML File

**File Name:**

`environmental_health_analyzer.html`

This HTML application was generated using Claude AI and includes the complete frontend (HTML, CSS, and JavaScript).
